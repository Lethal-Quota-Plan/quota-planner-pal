# Lethal Company Quota Calculator

**Lethal-Quota-Plan.github.io**  
A comprehensive, multi-featured web application for managing and planning quotas in the game "Lethal Company".

## Overview

This project offers an interactive quota calculator designed to help teams coordinate their efforts, maximize profits, and efficiently achieve company quotas within the gameplay. It serves both casual players and serious strategists who want to optimize their resource gathering and planning.

## Features

- **Quota Planning**: Track upcoming quotas and plan routes, loot distribution, and resource allocation. 75% confidence interval of each the moons' expected scrap value per day.
- **Profit Projection**: Forecast upcoming credit profit and losses; overtime bouns; and scrap left-overs based on the current data.
- **Customization**: Perform calculations as many weeks as you want, manually increase your luck value if you think you are more lucky.
- **History & Analytics**: View historical performance and analytics to improve decision-making.
- **Collaboration Tools**: Share plans and data with team members for effective group play.
- **User-Friendly Interface**: Responsive, accessible web interface built with modern web technologies.

## Usage

By simply input moons you want to visit by order for each week and the value of the scraps you will sold at each of week-ends, a weekly-ordered statistical matric graph will be calcuated and present for you.
